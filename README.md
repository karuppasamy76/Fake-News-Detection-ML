Fake News Detection using Machine Learning

This project detects whether a news article is fake or true using Natural Language Processing and Machine Learning techniques.

Technologies Used
Python
Pandas
Scikit-learn
TF-IDF Vectorization
Logistic Regression

Steps
1. Load dataset
2. Clean text data
3. Convert text to numerical vectors using TF-IDF
4. Train Logistic Regression model
5. Predict whether news is fake or true
